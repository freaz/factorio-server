# Factorio server

* https://hub.docker.com/r/dtandersen/factorio/
* https://github.com/dtandersen/docker_factorio_server

## Commands

```bash

# start server
$ bin/start

# stop server
$ bin/stop

# view logs
$ bin/logs

# back up data
$ bin/backup /full/path/to/backup

# restore data
$ bin/restore /full/path/to/backup

```
